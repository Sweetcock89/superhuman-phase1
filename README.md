# 🧬 SUPERHUMAN: PHASE 1
**Период:** 10.05.2026 – 13.06.2026 (35 дней)  
**Цель:** Стабилизация нейробиохимии, запуск когнитивного режима 100%, эмоциональная устойчивость, подготовка к отпуску.

## 📌 Правила использования
1. Каждый день открывайте файл `daily-plan/weekX.md`
2. Отмечайте выполнение в Loop Habit Tracker + ставьте галочку в Markdown
3. Вечером (22:00) заполняйте блок «📝 Выгрузка»
4. Тренировки: Пн/Ср/Пт 08:30–10:00. В остальные дни — активное восстановление.
5. Контакт с Ирой: стратегия «Маяк». Не инициировать. Отвечать кратко, тепло, без анализа отношений.

## 🚀 Синхронизация с Android
- Установите `Loop Habit Tracker` из F-Droid
- Импортируйте `config/loop_habits.csv`
- Отключите оптимизацию батареи для Loop
- Зафиксируйте расписание в системном календаре
